About Me

Hello! I am Michael. I am a passionate developer with a background in Customer Service, a love for video games and for exploring ways to express my creativity. My tech journey began when I decided to switch careers from delivering pizzas to pursuing a job in game development. I completed a 12-week Software Engineering Bootcamp with General Assembly in April of 2022. From there, I completed a 6-month QA Analyst Apprenticeship with Bounteous where I gained experience and an understanding of the SDLC within a cross-functional Agile environment. I currently spend my free time exploring game development using Blender and Unity3D. Thanks for taking the time to check out my projects!

Projects

Build-A-Farm

Build-A-Farm was a group project that took first place in a three day hackathon where the theme was accessibility in education technologies. We designed a math game for children to practice simple addition and subtraction. This project was built using React.js and incorporated simple game logic to provide feedback based on the users input.

Bad Interview

Bad Interview was full-stack solo project designed as an interview practice tool. Users could start sessions, choose a number of questions they would like to see, and input their answers. There were a few hurdles I had to overcome in this project. One example was I had to figure out how to ensure the user doesn't get the same question multiple times in a session. This application was built using a MERN stack.

Book Roulette

Book Roulette was a group based MERN stack project built in three days to provide users with a book recommendation tool. Users select the genre that they are interested in and a random book from the database is recommended. I worked on the front-end, developing several React components, writing JavaScript to provide the "random" functionality, and contributed heavily during the planning phase.

Team Deets

Team Deets is a match analysis tool for the computer game League of Legends. This React.js application utilizes the data from third-party APIs to dynamically render stat cards for every player in each match.

Baby Gandalph

Proudly presenting my very first project, Baby Gandalph. Baby Gandalph is a browser-based clicking game based off the classic "Tamagachi" game. Users input a bit of data to make their experience a bit more dynamic. They then have to keep their character from becoming too tired, or hungry while training their skills at the same time to level up and grow stronger. This application was made using an Object Oriented Programming approach, JQuery, HTML5 and CSS3.
